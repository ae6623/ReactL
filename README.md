# ReactL

Here's the Repo I've learned from React, and you can follow me from 0 to learn React.


## Version

* Nodejs v8.6
* Npm v5.3.0
* Reactjs 15.x -> 16.x later
* Webpack 3.x

## Start

```
cd ReactL
npm i
npm run hot
open chrome -> localhost:9002

```
