# ReactL

This repo is the study for ReactL-router-redux, you can use react from 0 to 1 with my repo.

## version
---- 
Nodejs v8.6
Npm v5.3.0
Reactjs 15.x -> 16.x later
Webpack 1.x -> 3.x later

```
cd ReactL
npm i
npm run hot

```
